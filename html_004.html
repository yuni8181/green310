테이블태그

웹문서에서 표를 만드는 요소
table 태그 안에 열과,행의 태그를 자식태그로 갖는다.
table 태그는 블럭요소이다.
행을 먼저만들고 행내부에 열을 자손자라 두어 내용을 작성한다.
table 태그는 선값이 기본 0으로 셋팅되어있어 border='1' 이라는 속성값과 속성명을 작성한다

  • -행요소 tr태그
  • 열요소 - td태그(내용작성태그)
  • 병합속성 - 세로병합 : rowspan='갯수' / 가로병합 :colspan='갯수'
  • 병합시주의사항 -
